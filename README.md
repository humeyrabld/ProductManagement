# Product Management API

ASP.NET Core Web API byggt med Clean Architecture.

## Tekniker

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* MediatR (CQRS)
* Repository Pattern
* AutoMapper
* FluentValidation
* JWT Authentication
* Swagger

## Arkitektur

* API
* Application
* Domain
* Infrastructure

## Funktioner

* CRUD för produkter
* Kategorier
* JWT Login
* Rollbaserad åtkomst

## Installation

1. Clone repository
2. Run migrations
3. Start API
4. Open Swagger

## Authentication

The API uses JWT authentication and role-based authorization.
